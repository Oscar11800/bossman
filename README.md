# Bossman

You talk to a computer. That computer talks to another computer. That computer builds things in a 3D world.

Inspired by Herbert Simon's 1961 prediction that computers would increasingly program themselves — Bossman takes that literally and makes it playable.

## How to Play

1. Open the site
2. **Hold SPACE** and say what you want built
3. Watch the Director (a snarky middle-manager) relay your order to the Coder (a retro computer)
4. The Coder writes and executes code live in the 3D scene
5. The Director roasts the result
6. Repeat. Go wild. Tell it to break things.

Drag, zoom, and pan to explore the 3D space. Everything the Coder builds is real and persistent until you refresh.

## Stack

Vite + TypeScript, Three.js, Web Speech API, Claude API (via Cloudflare Worker proxy), vanilla DOM. Hosted on GitHub Pages.
